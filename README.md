# Site Audit Service

Система автоматизированного аудита веб-сайтов для выявления проблем производительности, SEO и технического состояния.

## Описание проекта

Данный проект разработан в рамках производственной практики и предназначен для автоматизированной проверки веб-сайтов по заданному URL.  
Система анализирует основные характеристики сайта, включая техническое состояние, базовые SEO-параметры и показатели производительности, после чего формирует итоговую оценку и PDF-отчет.

Проект ориентирован на использование веб-разработчиками, SEO-специалистами, техническими специалистами и менеджерами веб-студии.

## Основные возможности

- запуск аудита по URL веб-сайта;
- автоматическая загрузка HTML-страницы;
- анализ базовых SEO-параметров:
  - title;
  - meta description;
  - H1;
  - canonical;
  - meta robots;
  - alt у изображений;
- анализ технического состояния:
  - HTTP-статус;
  - HTTPS;
  - редиректы;
  - robots.txt;
  - sitemap.xml;
  - битые ссылки;
- анализ базовой производительности:
  - время ответа;
  - размер HTML;
  - количество скриптов;
  - количество изображений;
- расчет итоговой оценки состояния сайта;
- сохранение истории аудитов в базе данных;
- генерация PDF-отчета;
- просмотр результатов через веб-интерфейс;
- работа через REST API.

## Архитектура проекта

Проект реализован как модульное backend-приложение с веб-интерфейсом.

### Основные модули

- `app/api/` — API и web-роуты;
- `app/models/` — модели базы данных;
- `app/schemas/` — Pydantic-схемы;
- `app/repositories/` — слой работы с базой данных;
- `app/services/` — бизнес-логика и анализаторы;
- `app/templates/` — HTML-шаблоны;
- `app/static/` — CSS и статические файлы;
- `app/utils/` — вспомогательные функции.

### Анализаторы

Система включает три основных анализатора:

- **SEO Analyzer** — проверка SEO-параметров страницы;
- **Technical Analyzer** — проверка технического состояния сайта;
- **Performance Analyzer** — проверка базовой производительности страницы.
